# 🚀 3D Personal Portfolio

> A modern, interactive **3D developer portfolio** built to showcase skills, projects, and creativity with immersive animations and smooth user experience.

## 📌 Overview

This project is a **next-generation developer portfolio** that goes beyond traditional static websites. It integrates **interactive 3D elements, smooth animations, and modern UI/UX practices** to create a visually engaging experience.

---

## ✨ Features

* 🎮 Interactive 3D experience
* ⚡ Smooth animations and transitions
* 📱 Fully responsive design
* 🧭 Clean and structured navigation
* 📂 Project showcase section
* 📬 Contact integration
* 🎨 Modern UI/UX design

---

## 🛠️ Tech Stack

### Frontend

* React.js
* JavaScript (ES6+)

### 3D & Animation

* Three.js
* React Three Fiber
* Framer Motion / GSAP

### Styling

* Tailwind CSS / CSS3

### Build Tools

* Vite / Webpack

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/AnasNihal/Personal-portfolio-3D.git
cd Personal-portfolio-3D
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Run development server

```bash
npm run dev
```

### 4️⃣ Open in browser

```
http://localhost:5173
```
